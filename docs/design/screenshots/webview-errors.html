에러·트러블슈팅 일지. 발생한 시점·원인·해결을 시간순(역시간순)으로 기록.

2026-05-30 — Go 바이너리 못 찾음 알림 재발

  • Status: IGNORED (Antigravity 자체 문제)
  • 원인: Antigravity의 내장 Go 관련 기능이 Go 1.x 설치 요구. 우리 extension은 TypeScript 기반이라 Go 사용 0%.
  • 해결: 무시. X로 닫음.

2026-05-22 — Extension Development Host에서 우리 extension 활성화 안 됨

  • Status: RESOLVED
  • 원인: activationEvents: ["workspaceContains:.blueprint/state.md"] 가 워크스페이스 폴더가 열려 있을 때만 트리거되는데, EDH가 폴더 없이 환영 화면 상태로 떠서 조건이 안 걸림.
  • 해결: .vscode/launch.json args 마지막에 "${workspaceFolder}" 추가.

2026-05-22 — Google.securecoder activation 실패 (Antigravity 내장)

  • Status: IGNORED (Antigravity 자체 문제, 우리 코드 무관)
  • 원인: Antigravity IDE의 내장 extension Google.securecoder 디렉터리에 out/extension.js 파일이 누락됨.
  • 해결: 우리가 고칠 수 없음. 다이얼로그를 X로 닫고 진행.
